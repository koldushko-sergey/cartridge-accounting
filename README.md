Cartridge QR Tracking System

Система учета картриджей для принтеров с использованием QR-кодов, Google Sheets и Google Apps Script.

Проект позволяет быстро отслеживать жизненный цикл каждого картриджа: от передачи на заправку до возврата в эксплуатацию.

Возможности:
Идентификация картриджа по QR-коду.
Автоматическое ведение журнала операций.
Изменение статуса картриджа одним нажатием.
Хранение истории всех операций.
Работа полностью в Google Sheets.
Не требует установки дополнительного ПО.

Как это работает:
Пользователь сканирует QR-код.
Открывается Web App Google Apps Script.
Определяется ID картриджа.
Пользователь выбирает действие:
Взят на заправку
Отдал в работу
Запись автоматически добавляется в журнал.
Статус картриджа обновляется.
